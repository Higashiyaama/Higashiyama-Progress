How the page is built

  • <!DOCTYPE html> tells the browser this is HTML5.
  • <html lang="en"> starts the page and sets the language.
  • <head> contains invisible setup like title and meta tags.
  • <title> shows on the browser tab.
  • <style> adds page design inside the same file.
  • <body> contains the visible content you read.

Visible content sections

Each part of your page is a section with a heading and text. This makes it easy to read.

<header>
    <h1>Page title</h1>
    <p>Short intro text</p>
</header>

<section>
    <h2>Section title</h2>
    <p>Paragraph text</p>
</section>

<footer>
    <p><a href="index.html">Back to home</a></p>
</footer>

You can copy that structure to your future pages. Just change the headings and text.
